# Image Compression Coursework

Курсовой проект по сжатию изображений на языке C.

В проекте планируется реализовать и сравнить два метода сжатия изображений:

- SVD-сжатие;
- JPEG-подобное сжатие на основе DCT и квантования.

Также будут реализованы метрики качества:

- PSNR;
- SSIM.

## Текущий этап

Создана начальная структура проекта и минимальная программа на C.
