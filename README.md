# Image Compression Coursework

Курсовой проект по сжатию изображений на языке C.

Тема проекта: реализация и сравнение методов сжатия изображений:
- SVD-сжатие;
- JPEG-подобное сжатие на основе DCT и квантования.

Также в проекте должны быть реализованы метрики качества PSNR и SSIM для сравнения исходного и восстановленного изображений.

## Что уже реализовано

На текущем этапе реализовано:

- начальная структура проекта;
- сборка через `Makefile`;
- обработка аргументов командной строки;
- чтение BMP-файлов;
- поддержка 24-bit BMP;
- сохранение BMP-файла;
- хранение изображения в структуре `Image`;
- преобразование изображения в оттенки серого;
- расчёт MSE;
- расчёт PSNR.
